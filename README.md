# Customer Shopping Behavior Analysis

This project analyzes a customer shopping behavior dataset from Kaggle using Python, PostgreSQL, SQL, and Power BI.

## Key Tasks
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* PostgreSQL Database Integration
* SQL Querying
* Power BI Dashboard Creation

## Tools Used
Python | Jupyter Notebook | PostgreSQL | pgAdmin 4 | SQL | Power BI

## Dataset
Customer Shopping Behavior Dataset (Kaggle)
